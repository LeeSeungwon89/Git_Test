# 깃 연습장
깃을 연습하는 저장소입니다.
불필요한 커밋은 삭제합니다.

# 깃 명령어

- 폴더에서 버전 관리 시작하기
```
git init
```

- 깃에 계정 연동하기
```
git config --global user.email "메일주소"
git config --global user.name "계정명"
```

- 커밋에 추가할 파일 선택하기
```
git add 파일명
```

- 커밋에 대한 상세설명 추가하기
```
# `m`은 massage의 약자입니다. 
git commit -m "설명"
```

- 커밋 목록 확인하기
```
git log
```
